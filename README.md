# 🚀 Node.js Backend Projects

This repository contains multiple backend projects built using Node.js and Express.

---

## 📂 Projects Included

### 1. Project 1 - API
- Basic REST API using Express
- Handles routes and requests

### 2. Project 2 - Authentication
- User login & signup system
- JWT based authentication

### 3. Project 3 - Blog API
- CRUD operations (Create, Read, Update, Delete)
- MongoDB integration

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- JavaScript

---

## ▶️ How to Run

1. Clone the repository
```
git clone <your-repo-link>
```

2. Go to project folder
```
cd project1-api
```

3. Install dependencies
```
npm install
```

4. Start server
```
npm start
```

---

## 📌 Note
Each project has its own folder and runs independently.
